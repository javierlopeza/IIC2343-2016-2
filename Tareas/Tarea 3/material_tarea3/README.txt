Para utilizar el Emulador del computador basico deberan seguir los siguientes pasos:

-Apretar el boton abrir para cargar el archivo con el codigo.
-Apretar Probar para ver como avanza el codigo.

Pueden hacer que la ejecucion sea mas lenta modificando el clock.

Ignore los .dlls que les piden ya que esos se supone que los terminaran haciendo ustedes en el proyecto.

En las pantallas del lado derecho podran ver como cambia la memoria RAM del computador, los registros, el PC y el stack, 
luego de apretar probar. Se pondra en negrita el codigo que se ejecuta actualmente.

El boton recargar es para volver a cargar el archivo.
